# 후원

`ai-engineering-from-scratch`는 무료 MIT 라이선스 커리큘럼입니다. 20개 phase에 걸쳐 428개 수업으로 구성되어 있습니다. 이 작업은 [Rohit Ghumare](https://github.com/rohitg00)가 만들고 유지관리합니다.

후원금은 수업을 배포하고, 사이트를 운영하고, issue queue에 답하는 데 필요한 시간을 지원합니다. 현금 후원만 받습니다. 크레딧 현물, 지분, 또는 "콘텐츠를 대신 써 주겠다"는 방식은 받지 않습니다. 아래 [강경 규칙](#강경-규칙)을 참고하세요.

개인 또는 회사가 이 커리큘럼을 지원하고 싶다면, 이 페이지가 요금표입니다.

## 후원 방법

- **GitHub Sponsors:** [github.com/sponsors/rohitg00](https://github.com/sponsors/rohitg00)

GitHub Sponsors가 결제, 영수증, 세금 양식을 처리합니다. [GitHub 정책](https://docs.github.com/en/sponsors/receiving-sponsorships-through-github-sponsors/about-github-sponsors-for-open-source-contributors)에 따라 개인 후원에는 플랫폼 수수료 0%, 조직 후원에는 최대 6%가 적용됩니다.

명확히 하자면, 이 커리큘럼에는 토큰, 코인, NFT, 지갑, 체인 파트너십이 없습니다. 관리자는 어떤 암호화폐 프로젝트도 보증하거나 홍보하거나 제휴하거나 결제를 받지 않습니다. 위의 후원 링크가 유일한 채널입니다. 이 프로젝트 이름을 사용하는 그 밖의 모든 것은 무관합니다.

## 강경 규칙

이 규칙은 협상 대상이 아닙니다. 받아들일 수 없는 후원사는 정중히 거절합니다.

1. **수업 본문 내 노출 금지.** 로고는 README 후원 블록, 커리큘럼 사이트의 후원 페이지, BACKERS.md에만 표시됩니다. `phases/**/docs/en.md`, `outputs/`, 코드 샘플, 또는 학습자가 커리큘럼 콘텐츠 자체를 읽는 곳에는 절대 넣지 않습니다.
2. **"Supported provider"는 "recommended"가 아닙니다.** 모든 API 수업은 같은 인터페이스 뒤에 세 개 이상의 provider를 보여 줍니다. 후원사는 다른 provider와 함께 나열될 뿐이며, 기본값, 선호 선택지, 또는 "무엇을 써야 하나요?"의 답으로 표시되지 않습니다.
3. **후원사 작성 콘텐츠 금지.** 모든 수업은 관리자가 작성합니다. 후원사는 통합 PR의 기술적 정확성만 검토합니다. 서사를 제안하거나, trade-off를 프레이밍하거나, 비교를 거부할 수 없습니다.
4. **로드맵 거부권 없음.** Platinum 후원사는 누구나 그렇듯 로드맵 제안을 제출할 수 있습니다. 무엇을 배포할지는 관리자가 결정합니다.
5. **편집 충돌 30일 종료.** 후원사가 콘텐츠 편향을 압박하면 후원은 30일 이내에 종료되고 비례 환불됩니다. 로고는 다음 사이트 배포에서 내려갑니다.
6. **충돌 거절.** 커리큘럼 원칙과 직접 충돌하는 제품의 후원은 거절합니다. 예: 폐쇄 루프 vibe-coding 도구, vendor lock-in 전도, 관측성을 무시하거나 열린 형식 없이 배포되는 agent 제품. 거절 여부는 전적으로 관리자의 재량입니다.
7. **현금만.** 크레딧 현물, 지분, 무료 하드웨어, "DevRel을 대신 해 주겠다"는 제안, 번들 거래는 받지 않습니다. 가치를 낮게 잡기 쉽고 깔끔하게 회계 처리하기 어렵기 때문입니다.

## 예비 후원사의 역제안

회사에 다른 요청이 있다면, 먼저 등급 체계와 강경 규칙을 읽고 첫 이메일에서 구체적인 등급과 기간을 제안하세요. "무료 크레딧과 hero placement를 교환하면 어떨까요" 또는 "통합 내용을 저희가 직접 쓰고 싶습니다"로 시작하지 마세요. 이런 요청은 위 강경 규칙에 따라 사전에 거절되며, 답장은 이 페이지 링크로 끝납니다.

## 가격 기준점

위 등급 금액은 (a) 비교 가능한 오픈소스 프로젝트의 공개 후원 페이지와 (b) 월 5만~10만 방문자 규모 개발자 매체의 표준 후원 슬롯 단가를 기준으로 삼았습니다. 2026-05에 확인했습니다.

비교 가능한 오픈소스 요금표:

- **오픈소스 기준선** — [Drupal AI Developer Assistant](https://opencollective.com/drupal-ai-initiative/projects/aidev), [Babel](https://opencollective.com/babel), [Parcel](https://opencollective.com/parcel), [Vue.js](https://opencollective.com/vuejs)는 모두 월 $100의 Bronze를 텍스트 인정으로 시작합니다. 이 커리큘럼의 Bronze가 $250인 이유는 개별 repo가 갖지 못한 audience traffic을 커리큘럼이 갖고 있기 때문입니다.
- **$750 Silver**는 Babel Silver($500)와 Drupal AI Gold($500)보다 높고, Vue Platinum($2,000)보다 낮습니다. 커리큘럼의 월간 traffic 기준으로 방어 가능한 가격입니다.
- **$2,000 Gold**는 Babel Base Support(연 $24K 청구 = 월 $2K)와 Vue Platinum에 맞춥니다.
- **$5,000 Platinum**은 Vue Diamond에 맞춥니다. 7.5K stars + 월 55K 방문자 + 현재 성장 기울기에서는 dedicated lesson + hero placement가 이 가격을 정당화합니다.
- **Diamond / Title($10K+)**은 건너뜁니다. 월 방문자가 250K를 넘으면 다시 검토할 수 있습니다.

## 후원금 사용처

다음 1달러가 쓰이는 순서대로 나열합니다.

1. 새 수업과 issue queue에 쓰는 관리자 시간.
2. 사이트 호스팅, 도메인, CDN(Vercel + custom domains).
3. 다이어그램 제작 도구, 폰트 라이선스, 디자인 자산.
4. phase가 관리자의 전문 범위를 벗어나는 영역을 다룰 때 guest lesson reviewer에게 지급하는 일회성 리서치 또는 콘텐츠 비용.
5. 30일 넘게 열려 있는 특정 issue에 대한 기여자 bounty.

## 후원사 되기

1. 위 등급 중 하나를 선택합니다.
2. [GitHub Sponsors](https://github.com/sponsors/rohitg00)에서 구독합니다.
3. Silver 이상은 관리자에게 로고(SVG 권장), 연결할 URL, 약정 기간을 이메일로 보냅니다.
4. 로고는 보통 48시간 이내 다음 사이트 배포에 반영됩니다.
5. 영수증과 인보이스는 GitHub Sponsors가 자동 발행합니다.

## 후원 종료하기

취소는 GitHub Sponsors dashboard에서 한 번 클릭하면 됩니다. 현재 결제 기간이 끝난 뒤 다음 사이트 배포에서 로고가 내려갑니다. 환수도, 종료 인터뷰도, 감정 상할 일도 없습니다. 후원은 커리큘럼을 지원하는 것이지 관계를 사는 것이 아닙니다.
