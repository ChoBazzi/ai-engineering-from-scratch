이 프로젝트 소개

AI Engineering from Scratch는 핵심 AI 알고리즘을 모두 직접 구현해 보는 무료 오픈소스 커리큘럼입니다. 선형대수부터 autonomous agent까지 20개 phase, 503개 lesson을 Python, TypeScript, Rust, Julia로 다룹니다.

왜 존재하는가

대부분의 AI 자료는 흩어진 조각으로 가르칩니다. 여기에는 논문, 저기에는 fine-tuning 글, 또 다른 곳에는 framework demo가 있습니다. loss curve를 설명하지 못해도 chatbot을 배포할 수 있고, 호출하는 모델 내부에서 attention이 무엇을 하는지 몰라도 agent에 tool을 연결할 수 있습니다.

이 커리큘럼은 그 등뼈입니다. 모든 알고리즘을 먼저 원시 수학에서 직접 작성한 다음, production library로 같은 작업을 실행해 library가 무엇을 하고 있었는지 보게 합니다. PyTorch가 등장할 때쯤이면 이미 그것이 무엇을 계산하는지 알고 있습니다. 각 lesson은 prompt, skill, agent, MCP server 같은 재사용 가능한 artifact로 끝납니다.

어떻게 만들어지는가

lesson은 AI의 도움을 받아 작성하고, 사람이 primary source와 대조해 검토합니다. lesson이 사실을 말하는 곳에서는 2차 요약이 아니라 원문 RFC, spec, 연구 논문을 인용합니다. 수정 제안은 환영하며 GitHub에서 공개적으로 추적합니다.

사이트 자체는 의도적으로 단순합니다. framework 없이 손으로 쓴 HTML, CSS, vanilla JavaScript입니다. 단일 build script(site/build.js)가 repository의 lesson Markdown을 읽고 매 deploy마다 catalog, search index, sitemap, llms.txt를 생성하므로 공개된 숫자가 source와 어긋날 수 없습니다. 호스팅은 Vercel에서 합니다.

누가 만드는가

Rohit Ghumare와 contributors가 유지 관리합니다. MIT license이며 영원히 무료입니다. token도, course upsell도, gated content도 없습니다.
